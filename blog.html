Important FAQ

What is the Document Object Model (DOM)?

The DOM is a structured representation of HTML elements in browsers.

How do you select an element from the DOM?

To select a DOM element, use methods like document.getElementById(), document.getElementsByClassName(), or document.querySelector().

What is event delegation in the context of the DOM, and why is it useful?

Event delegation uses a parent to handle events for child elements efficiently.

How do you manipulate an element's attributes and styles using the DOM?

Use setAttribute() for attributes and style.property for inline styles.
